LiteEditor 소개

LiteEditor는 웹 페이지에 쉽게 통합할 수 있는 고정형 리치 텍스트 에디터 플러그인입니다. 사용자가 다양한 서식을 적용할 수 있는 직관적인 도구 모음을 제공합니다.

주요 기능

  • 간단한 통합: 어떤 웹사이트에도 손쉽게 추가할 수 있습니다.
  • 플러그인 구조: 필요한 기능만 선택하여 사용할 수 있습니다.
  • 웹 표준 준수: JS와 CSS만으로 구현되어 호환성이 뛰어납니다.
  • 반응형 디자인: 모든 화면 크기에 맞게 조정됩니다.

지원 서식

  • undo: 실행 취소.
  • redo: 재실행.
  • 폰트: 지정된 텍스트에 폰트를 적용합니다.
  • 폰트 컬러지정한 컬러를 폰트에 적용합니다.
  • 제목 (Heading): 제목 텍스트 스타일을 적용합니다.
  • 하이라이트지정한 텍스트에 하일라이트 처리 합니다.
  • 굵게 (Bold): 텍스트를 굵게 표시합니다.
  • 기울임 (Italic): 텍스트를 기울임꼴로 표시합니다.
  • 밑줄 (Underline): 텍스트에 밑줄을 추가합니다.
  • 취소선 (Strike): 텍스트에 취소선을 추가합니다.
  • 코드 (Code): 텍스트를 코드 블록으로 형식화합니다.
  • 코드블럭 (CodeBlock): 선택한 텍스트를 코드 블록으로 형식화합니다.
  • 링크 (Link): 선택한 텍스트에 하이퍼링크를 추가합니다.
  • 이미지 (Image): 선택한 텍스트에 이미지를 추가합니다.
  • 테이블 (Table): 텍스트를 테이블로 형식화합니다.
  • 서식 지우기 (Reset Format): 모든 서식을 제거합니다.

기능 목록

LiteEditor는 플러그인 기반 구조로 다양한 기능을 확장할 수 있습니다.

format_bold 기본 서식

기본적인 텍스트 서식 기능을 제공합니다: 굵게, 기울임, 밑줄, 취소선 등

title 제목 스타일

H1부터 H6까지 다양한 크기의 제목 태그를 적용할 수 있습니다.

format_color_text 폰트 관련

다양한 폰트 패밀리와 색상, 하이라이트를 적용할 수 있습니다.

format_align_center 정렬 및 구조

텍스트 정렬, 들여쓰기, 블록쿼트, 목록 등 다양한 구조를 적용할 수 있습니다.

format_clear 서식 초기화

적용된 모든 서식을 제거하고 순수 텍스트만 유지하는 기능을 제공합니다.

사용 방법

다음과 같은 하나의 스크립트만으로 LiteEditor를 추가할 수 있습니다.

<!-- CSS 통합 로더 -->
<link rel="stylesheet" href="path/to/lite-editor/css/loader.css">

<!-- 에디터가 표시될 영역 -->
<div id="my-editor" contenteditable="true"></div>

<!-- JS 로더 -->
<script src="path/to/lite-editor/js/loader.js"></script>

<!-- 초기화 코드 -->
<script>
  document.addEventListener('lite-editor-loaded', function() {
    const editor = LiteEditor.init('#my-editor', {
      plugins: ['bold', 'italic', 'underline', 'link', 'reset'],
      dividers: [3] // 구분선 위치
    });
  });
</script>
